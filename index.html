Cougars Fleet: Autonomous Marine Robotics

Developed a fleet of low-cost, open-source Autonomous Underwater Vehicles (AUVs) for research in GPS-denied underwater localization using acoustic signals. Served as Research Assistant & Lead Systems Developer, architecting navigation systems and designing custom electronics.

C++ ROS 2 Python Linux KiCad Raspberry Pi Teensy 4.1
Challenge: Replace overly complex MOOS-IvP framework and design reliable underwater navigation system.
Solution: Engineered complete waypoint navigation system in C++ using ROS 2, wrote custom drivers for DVL and RF modules, designed main power distribution PCB.
Result: Improved navigation consistency, simplified mission creation, and enhanced system reliability while co-leading the project team.
Code

Agricultural Robotics Competition (ASABE)

Led software development for autonomous robot designed for the American Society of Agricultural and Biological Engineers (ASABE) student competition. Robot navigates arena to collect and sort eggs using advanced sensor fusion and real-time control systems.

Python ROS 2 PID Control Raspberry Pi 5 Time-of-Flight Sensors OpenCV Git
Challenge: Lead 3-person software team to create precise autonomous navigation in competitive arena environment.
Solution: Developed PID controllers using 4 ToF distance sensors for closed-loop control, architected ROS 2 software system, and wrote low-level hardware interfaces for motor drivers.
Result: Robot accurately centers itself and follows paths within arena walls with real-time sensor feedback and reliable hardware control.

About Me

My journey into engineering began with a curiosity for how things work, but it has evolved into a passion for making things work intelligently. I am a systems-thinker and a hands-on builder who thrives on the challenge of bridging the gap between hardware and software.

My experience has given me a unique, full-stack perspective. I've designed circuit boards in KiCad and written the low-level C++ drivers to control them. I've architected complex autonomous navigation systems in ROS 2 and developed control algorithms to guide robots through dynamic environments. This foundation in robotics has been the perfect training ground for mastering fundamental skills in system design, embedded programming, and real-time software development.

Beyond robotics, I am fascinated by the power of automation to create efficiency and solve problems. I actively explore technologies like LLMs, APIs, and platforms like n8n to build intelligent workflows, from personal assistants to automated business solutions.

I am now looking to apply my expertise in system integration and autonomous logic to new domains. I am particularly excited by the advancements in computer vision and machine learning and am actively seeking opportunities where I can contribute to building the next generation of smart, data-driven systems.
